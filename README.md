# Alacritty Config

## Features

- **Shell:** Zsh
- **Font:** JetBrainsMono Nerd Font
- **Font size:** 10 (with slight vertical offset for readability)
- **Colors:** Vibrant Nord-inspired palette
- **Opacity:** 0.8
- **Scrolling:** 10k history with smooth multiplier
- **Clipboard:** System clipboard + middle-click paste
- **Search & Vi mode:** Enabled
- **Live reload:** On

## Requirements

- [Alacritty](https://github.com/alacritty/alacritty)
- [Zsh](https://www.zsh.org)
- [JetBrainsMono Nerd Font](https://www.nerdfonts.com/font-downloads)
  `sudo pacman -S ttf-jetbrains-mono-nerd`

## Install

```bash
rm -rf ~/.config/alacritty
git clone https://github.com/norad32/alacritty-config.git ~/.config/alacritty
```
